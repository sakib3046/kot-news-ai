## Deployment Steps

### 1. Environment Setup
Copy .env.local template and fill in:
- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- CRON_SECRET

### 2. Pre-Deploy Verification
`ash
npm install
npm run build
npm audit
`

Expected output:
- Build: "Compiled successfully"
- TypeScript: "Finished TypeScript in X.Xs"
- Audit: "found 0 vulnerabilities"

### 3. Deploy
`ash
npm run start
`

### 4. Verify Routes
`ash
curl http://your-domain.com/
curl http://your-domain.com/admin/login
curl http://your-domain.com/api/auth/me
`

All should respond with 200 OK

### 5. Monitor
- Check error logs for any issues
- Monitor rate limit hits
- Track API response times
- Set up error tracking (Sentry)

## Rollback Plan

If deployment fails:
1. Stop current deployment
2. Restore from previous version
3. Check error logs
4. Review recent changes
5. Contact development team
